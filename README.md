# Vehicle API

This is the back-end for vehicle maintenance [web](https://github.com/Torttuga/vehiculo-front)

## Description
API URL: https://vehiculo-web-service.herokuapp.com/mantenimientos

[Nest](https://github.com/nestjs/nest) API with MongoDB

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
